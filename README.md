# Java-Note仓库

## 1、说明文档

编程常用技术栈进行归档整理，划分如下几个类目：【Jvm虚拟机、Java基础、设计模式、结构与算法、架构设计、数据服务、Spring系列、中间件系列、大数据系列、数据库系列、Linux系列】，后续持续维护管理。

详细内容，见→<b> [Wiki](https://gitee.com/cicadasmile/butte-java-note/wikis/pages)</b> 文档，编程是一个持续积累的过程。
